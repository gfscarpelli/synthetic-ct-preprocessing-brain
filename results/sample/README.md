# Sample sCT Results

These images show qualitative comparisons between reference CT 
and generated synthetic CT (sCT) across axial, coronal, and 
sagittal planes for four representative patients.

## Image source
CT images are derived from the SynthRAD2023 dataset:
Thummerer et al. (2023). [SynthRAD2023 Grand Challenge](https://doi.org/10.1002/mp.16529)

sCT images were generated as part of this thesis work.
